# Celeste Unity Port

This is a Unity 2021 LTS port of the original Celeste source code while maintaining full compatibility with the original XNA Framework, Monocle engine, MonoGame, and FNA implementations. The project integrates modern Unity features with classic Celeste gameplay systems.

## üéÆ Unity Features

- **Unity 6 LTS Compatible:** Full compatibility with Unity 2021 LTS (2021.3.45f2 or later)
- **Multi-Framework Support:** Maintains XNA, Monocle, MonoGame, and FNA compatibility
- **Asset Pipeline:** Converts Content to Unity Assets (Scenes, Sprites, Audio)
- **Unity Integration:** Optional Unity physics, input, and rendering
- **Modding Support:** Compatible with EverestAPI for modding
- **Map Editor:** Unity-based map editor with Loenn compatibility (Beta)
- **Physics Integration:** Optional Unity physics mode
- **Room Wrapper System:** Seamless room transitions with loading/unloading
- **Celeste Camera:** Advanced camera system with look-ahead
- **Tweening System:** High-performance animation system
- **3D Overworld:** Mountain, bird, moon, building, and heart 3D models
- **Framework Conversion:** Automatic conversion between Monocle/XNA/MonoGame/FNA and Unity
- **BepInEx Integration:** Unity-specific modding support
